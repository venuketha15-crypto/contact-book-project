# 📒Contact Book 

A responsive Contact Book web application built using HTML, CSS, and JavaScript.

## 🚀 Live Demo


## ✨ Features

- Add Contacts
- Edit Contacts
- Delete Contacts
- Search Contacts
- Sort Contacts
- Mark Contacts as Favorites ⭐
- Contact Categories
  - Family
  - Friends
  - Work
  - Emergency
  - Others
- Local Storage Support
- Responsive Mobile Design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
